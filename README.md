# 📊 HR Analytics Dashboard

An interactive HR Analytics Dashboard built using Streamlit and Machine Learning for candidate placement analysis, job acceptance prediction, and recruitment insights.


# 🚀 Features

* Total Candidates KPI
* Placement Rate Analysis
* Job Acceptance Rate
* Average Interview Score
* Skills Match Percentage
* Offer Dropout Rate
* High-Risk Candidate Detection
* Interactive Dashboard Visualizations
* Dataset Preview


# 🛠️ Technologies Used

* Python
* Pandas
* Streamlit
* Machine Learning
* Data Analytics
  

# 📈 KPIs Included

* 👨‍💼 Total Candidates
* ✅ Placement Rate
* 📌 Job Acceptance Rate
* 🎤 Average Interview Score
* 🧠 Average Skills Match %
* ⚠️ Offer Dropout Rate
* 🚨 High-Risk Candidate Percentage


# 📊 Dashboard Preview

Interactive visualizations for HR recruitment analytics and placement prediction.



# 🎯 Project Goal

To analyze candidate recruitment data and build an interactive dashboard that helps HR teams understand placement trends, candidate performance, and hiring risks.
